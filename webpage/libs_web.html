Instructions to Change System Language to Japanese

  1. Search for Language Settings: 
    • In the Windows Search Box (⌘ + S), search for Language.
  2. Open Language Settings: 
    • From the search results, select Language settings under the Settings menu.
  3. Install Japanese Language: 
    • In the Preferred Languages section, click on Add a language.
                    
    • Search for Japanese and follow the prompts to install it.
  4. Access Administrative Language Settings: 
    • In the same Language settings menu, navigate to Related settings.
                    
    • Click on Administrative language settings.
  5. Change System Locale: 
    • In Administrative, under the Language for non-Unicode programs section, click on Change system locale.
                    
    • From the dropdown menu, select Japanese (Japan).
  6. Restart Your PC: 
    • Confirm the changes and restart your computer for the changes to take effect.
